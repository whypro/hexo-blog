一个拐角检测

一个拐角检测可分解为水平检测和垂直检测，当两个同时满足时，便两点可通过一个拐角相连。即：

一个拐角检测 = 水平检测 && 垂直检测

A 点至 B 点能否连接可转化为满足任意一点：

  1. A 点至 C 点的垂直检测，以及 C 点至 B 点的水平检测；

  2. A 点至 D 点的水平检测，以及 D 点至 B 点的垂直检测。
